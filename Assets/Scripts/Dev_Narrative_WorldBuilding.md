## 1. 세계관 (World Building)

### 1.1 세계 설정: 크리스탈리아(Crystallia)

**크리스탈리아**는 6개의 보석 원소로 이루어진 신비로운 대륙입니다. 이 세계는 육각형 크리스탈 격자로 구성되어 있으며, 각 육각형은 세계의 에너지를 담는 그릇 역할을 합니다.

#### 세계의 구조
- **헥사코어 그리드(Hexacore Grid)**: 세계를 구성하는 육각형 에너지 격자
  - 각 육각형 칸은 "에너지 슬롯"으로, 보석의 힘을 담을 수 있음
  - 격자는 세계의 균형을 유지하는 자연의 시스템
  - 같은 색 보석이 3개 이상 모이면 "공명(Resonance)" 현상이 일어나 에너지가 정화됨

#### 보석의 의미
6가지 원소 보석은 세계의 생명력 그 자체입니다:

| 보석 | 원소 | 상징 | 역할 |
|------|------|------|------|
| **루비(Ruby)** | 불 | 열정과 용기 | 세계의 온기와 활력 |
| **사파이어(Sapphire)** | 물 | 평온과 지혜 | 세계의 균형과 흐름 |
| **에메랄드(Emerald)** | 자연 | 생명과 치유 | 세계의 생명력 |
| **앰버(Amber)** | 빛 | 희망과 기쁨 | 세계의 빛과 낙관 |
| **자수정(Amethyst)** | 신비 | 마법과 직관 | 세계의 마법 에너지 |
| **시트린(Citrine)** | 대지 | 안정과 풍요 | 세계의 기반과 안정 |

#### 세계의 위기
어느 날, **카오스 에너지(Chaos Energy)**가 크리스탈리아를 침범하기 시작했습니다.
- 육각형 격자 곳곳에 **오염체(Corruption)**들이 기생하기 시작
- 보석들의 공명이 방해받아 에너지가 왜곡됨
- 세계의 균형이 무너지면서 빛이 사라지고 있음

### 1.2 플레이어의 목표

플레이어는 **크리스탈 수호자(Crystal Guardian)**가 되어:
1. 보석들을 올바르게 정렬하여 **공명 정화(Resonance Purification)**를 일으킴
2. 각 지역의 헥사코어 그리드에서 오염체들을 제거
3. 100개 지역을 정화하여 크리스탈리아의 빛을 되찾음

## 2. 주인공 캐릭터

### 2.1 루나 스타브라이트(Luna Starbright)

<img src="concept_luna.png" width="300" alt="Luna Concept Art"/>

#### 기본 정보
- **나이**: 18세
- **직업**: 견습 크리스탈 수호자
- **성격**: 밝고 긍정적, 호기심 많음, 약간 덜렁대지만 책임감 강함
- **외형**:
  - 파란색-보라색 그라데이션 머리카락 (크리스탈 에너지 영향)
  - 육각형 펜던트를 목에 착용 (수호자의 증표)
  - 가벼운 원피스 스타일의 마법사 의상

#### 배경 스토리
루나는 크리스탈리아 변방 마을에서 자란 평범한 소녀였습니다. 어느 날 숲에서 빛나는 육각형 크리스탈을 발견하고 만진 순간, 그녀는 보석들의 "공명"을 느낄 수 있는 능력이 깨어났습니다.

크리스탈 아카데미의 현자 오라클리온이 찾아와 말했습니다:
> "네가 바로 예언된 새로운 수호자다. 네게는 보석들의 조화를 회복할 특별한 재능이 있어."

#### 동기
- **개인적 동기**: 자신의 고향 마을도 카오스에 오염되기 전에 세계를 구하고 싶음
- **성장 목표**: 미숙한 견습생에서 진정한 크리스탈 수호자로 성장
- **비밀스러운 꿈**: 전설의 "레인보우 크리스탈"을 찾아 모든 원소를 마스터하고 싶음

#### 게임 내 역할
- 플레이어의 아바타로, 육각형 그리드를 회전시키는 마법을 사용
- 스토리 진행 중 각 지역의 오염체들과 대면하며 성장
- 특수 보석(드릴, 폭탄, 레이저 등)은 루나가 습득한 "상급 정화 기술"

## 3. 적군 세력 배경

### 3.1 적군 세력의 기원

**카오스 로드 엔트로피(Chaos Lord Entropy)**가 이끄는 **오염 군단(Corruption Legion)**이 적군의 정체입니다.

#### 엔트로피의 목적
- 크리스탈리아의 완벽한 조화를 "지루하다"며 증오
- 무질서와 혼돈이야말로 진정한 자유라고 믿음
- 모든 보석을 무색으로 만들어 "완벽한 무(無)"를 달성하려 함

### 3.2 10종 적군 - 오염체들의 설정

각 적군은 엔트로피가 만들어낸 **오염의 화신**들입니다.

#### 1. 색상 도둑 (Chromophage / 크로모파지)
```
티어: 1 (초반 적)
외형: 회색 안개 형태의 기생체
특징: 촉수처럼 뻗어 보석의 색을 빨아들임
```
**스토리**: 가장 기초적인 오염체. 보석에 기생하여 색을 흡수해 무색 돌로 만듭니다.
- **대사**: "색...색이 필요해... 더 많은 색을..."

#### 2. 속박의 사슬 (Chain Anchor / 체인 앵커)
```
티어: 2
외형: 검은 쇠사슬로 육각형을 휘감은 모습
특징: 육각형 칸을 고정시켜 움직임 차단
```
**스토리**: 엔트로피의 간수. 보석들이 공명하지 못하도록 격자를 물리적으로 봉쇄합니다.
- **대사**: "움직임은 혼돈의 시작... 너희는 여기 갇혀야 해."

#### 3. 가시 기생충 (Thorn Parasite / 쏜 패러사이트)
```
티어: 2
외형: 보석 표면에 검은 가시가 돋아난 형태
특징: 제거 시 주변에 가시를 뿌려 피해를 줌
```
**스토리**: 방어 본능이 강한 오염체. 자신을 제거하려는 수호자에게 저주를 걸어 대가를 치르게 합니다.
- **대사**: "날 건드리면 후회할 거야... 크흡!"

#### 4. 분열체 (Divider / 디바이더)
```
티어: 3
외형: 반투명한 젤리 형태, 중앙에 어두운 핵
특징: 제거되면 2개로 분열하여 재생성
```
**스토리**: 생명력이 강한 오염체. 한 번 정화해도 즉시 복제되어 끈질기게 저항합니다.
- **대사**: "하나를 없애면 둘이 된다... 영원히!"

#### 5. 중력 왜곡자 (Gravity Warper / 그래비티 워퍼)
```
티어: 4
외형: 블랙홀처럼 주변을 왜곡시키는 구체
특징: 위쪽 블록의 낙하를 차단
```
**스토리**: 중력을 조종하는 오염체. 보석들이 자연스럽게 흐르는 것을 막아 격자의 흐름을 차단합니다.
- **대사**: "여기서는 내가 중력이다... 낙하 따윈 없어."

#### 6. 반사 장막 (Reflection Shield / 리플렉션 실드)
```
티어: 4
외형: 보석을 둘러싼 육각형 결계막
특징: 특수 블록 효과를 1회 무효화
```
**스토리**: 엔트로피가 만든 방어 마법. 수호자의 강력한 정화 기술을 반사하여 무력화시킵니다.
- **대사**: "네 마법은 나에게 통하지 않아... 반사!"

#### 7. 시간 정지자 (Time Freezer / 타임 프리저)
```
티어: 5
외형: 얼음 시계 형태, 초침이 멈춰 있음
특징: 회전 시 2턴 소모
```
**스토리**: 시간을 느리게 만드는 오염체. 수호자의 행동을 둔화시켜 시간과 에너지를 2배로 소모하게 만듭니다.
- **대사**: "시간은 내 편이야... 천천히, 아주 천천히..."

#### 8. 공명 트윈 (Resonance Twin / 레조넌스 트윈)
```
티어: 5
외형: 두 개가 빛의 선으로 연결된 쌍둥이 보석
특징: 둘 다 동시에 제거해야 함
```
**스토리**: 완벽한 동조 상태의 오염체 쌍둥이. 하나만 정화하면 남은 하나가 즉시 부활시킵니다.
- **대사**: "우린 하나야... 함께 사라지거나, 함께 살아남거나."

#### 9. 어둠의 포자 (Shadow Spore / 섀도우 스포어)
```
티어: 6
외형: 검은 버섯처럼 생긴 포자체
특징: 3턴마다 주변 칸으로 확산
```
**스토리**: 가장 위험한 감염체. 방치하면 빠르게 증식하여 그리드 전체를 오염시킵니다.
- **대사**: "번식...확산... 이 세계는 곧 어둠에 뒤덮일 것이다..."

#### 10. 카오스 군주 (Chaos Overlord / 카오스 오버로드)
```
티어: BOSS
외형: 엔트로피의 화신, 무지갯빛이 왜곡된 거대한 결정체
특징: 랜덤하게 위 능력들을 조합하여 사용
```
**스토리**: 엔트로피가 각 지역의 마지막 거점에 배치한 보스급 오염체. 모든 오염 능력을 복합적으로 사용합니다.
- **대사**: "질서는 환상이다... 진정한 힘은 혼돈에서 나온다!"

### 3.3 적군 조직 구조

```

    │
    ├─ 고위 오염체 (Tier 5-6)
    │   ├─ 시간 정지자
    │   ├─ 공명 트윈

    │   ├─ 중력 왜곡자
    │   └─ 반사 장막
    │
    └─ 하급 오염체 (Tier 1-2)
        ├─ 색상 도둑
        ├─ 속박의 사슬
        └─ 가시 기생충
```

## 4. 스토리 진행 구조

### 4.1 챕터 구성 (100 스테이지 = 10 챕터 x 10 스테이지)

각 챕터는 크리스탈리아의 한 지역을 대표하며, 독특한 배경과 테마를 가집니다.

#### 챕터별 구조
- **스테이지 1-8**: 일반 스테이지 (난이도 점진적 상승)
- **스테이지 9**: 미니보스 스테이지 (특수 오염체 2종 동시 등장)
- **스테이지 10**: 챕터 보스 스테이지 (카오스 오버로드 등장)

### 4.2 10개 챕터 테마

#### Chapter 1: 크리스탈 숲 (Crystal Forest) - 스테이지 1-10
```
배경: 루나의 고향 근처 숲. 푸른 에메랄드 나무들이 빛나는 평화로운 지역
테마: 튜토리얼 + 초보 수호자의 첫 시험
주요 적: 색상 도둑, 속박의 사슬
보스: 크로모파지 킹 (거대한 색상 도둑)
스토리: 루나가 수호자로서 첫 임무를 수행하며 기본 정화 기술을 익힘
```

#### Chapter 2: 사파이어 호수 (Sapphire Lake) - 스테이지 11-20
```
배경: 잔잔한 푸른 호수, 물 위에 떠 있는 육각형 크리스탈 섬들
테마: 물의 흐름과 균형
주요 적: 색상 도둑, 가시 기생충
보스: 아쿠아 체인 (물을 얼리는 속박의 사슬 변종)
스토리: 루나가 물의 원소를 이해하고 "흐름에 맡기는" 지혜를 배움
```

#### Chapter 3: 루비 화산 (Ruby Volcano) - 스테이지 21-30
```
배경: 활발한 화산 지대, 용암이 흐르는 틈 사이 빛나는 루비들
테마: 열정과 도전
주요 적: 가시 기생충, 분열체
보스: 인페르노 디바이더 (분열할 때마다 불기둥을 만드는 보스)
스토리: 루나가 용기를 내어 위험을 극복하며 자신감을 얻음
```

#### Chapter 4: 앰버 초원 (Amber Meadow) - 스테이지 31-40
```
배경: 끝없이 펼쳐진 황금빛 초원, 따뜻한 햇살이 내리쬐는 평화로운 지역
테마: 희망과 회복
주요 적: 분열체, 중력 왜곡자
보스: 그라비티 타이탄 (거대한 중력 왜곡자, 보석을 공중에 띄움)
스토리: 루나가 지친 마음을 회복하고 여정을 계속할 힘을 얻음
```

#### Chapter 5: 자수정 동굴 (Amethyst Cavern) - 스테이지 41-50
```
배경: 신비로운 지하 동굴, 보라빛 크리스탈이 벽면을 뒤덮음
테마: 신비와 마법
주요 적: 중력 왜곡자, 반사 장막
보스: 미러 메이지 (반사 장막의 주인, 루나의 기술을 복사함)
스토리: 루나가 고급 마법 기술을 익히며 자신의 한계를 넘어섬
```

#### Chapter 6: 시트린 사막 (Citrine Desert) - 스테이지 51-60
```
배경: 드넓은 사막, 모래 속에 묻힌 오렌지빛 크리스탈들
테마: 인내와 끈기
주요 적: 반사 장막, 시간 정지자
보스: 크로노스 샌드 (시간을 되돌리는 모래시계 형태 보스)
스토리: 루나가 고난 속에서도 포기하지 않는 진정한 끈기를 배움
```

#### Chapter 7: 무지개 정원 (Rainbow Garden) - 스테이지 61-70
```
배경: 모든 색 보석이 조화롭게 핀 환상적인 정원
테마: 조화와 균형
주요 적: 시간 정지자, 공명 트윈
보스: 트윈 하모니 (완벽하게 동조된 6쌍의 공명 트윈들)
스토리: 루나가 모든 원소의 조화를 이해하며 한 단계 성장
```

#### Chapter 8: 폭풍의 탑 (Storm Tower) - 스테이지 71-80
```

### 4.3 스토리 전달 방식

#### 컷씬 (Cut-scene)
- **챕터 시작**: 각 챕터 첫 스테이지 전 짧은 애니메이션 컷씬 (15-20초)
  - 새로운 지역 도착 장면
  - 루나와 조력자의 대화
  - 해당 챕터 테마 소개

- **보스 등장**: 스테이지 10 시작 전 보스 등장 연출 (10초)
  - 보스의 위협적인 등장
  - 루나의 결의 표정

- **챕터 클리어**: 보스 처치 후 승리 컷씬 (10초)
  - 지역이 정화되며 빛을 되찾는 장면
  - 루나의 성장 모먼트

#### 대화 시스템
- **스테이지 시작 전**: 짧은 대화창 (2-3줄)
  - 조력자 NPC의 조언이나 격려
  - 루나의 각오 한 마디

- **스테이지 중간**: 특수 이벤트 발생 시
  - 새로운 적 등장 시 짧은 설명
  - 위기 상황에서 조력자의 힌트

#### UI 스토리텔링
- **로딩 화면**: 크리스탈리아의 세계관 설명 (Tip 형식)
- **스테이지 선택 맵**: 이미 클리어한 지역은 밝게 빛나며 복구된 모습 표시
- **챕터 맵**: 10개 지역이 크리스탈리아 전체 지도에 표시되어 진행도 시각화

## 6. 내러티브와 게임플레이 연결

### 6.1 핵심 연결 고리: "공명 정화 의식"

게임의 모든 플레이는 **공명 정화 의식(Resonance Purification Ritual)**이라는 세계관적 행위입니다.

### 6.2 게임 메카닉과 세계관 연결표

| 게임 메카닉 | 세계관적 의미 | 비주얼/사운드 연출 |
|------------|--------------|------------------|
| **육각형 그리드** | 세계를 구성하는 헥사코어 격자 | 격자선이 에너지처럼 빛남 |
| **블록 회전** | 루나가 마법으로 격자를 회전시킴 | 회전 시 마법진 이펙트 |
| **3개 매칭** | 같은 원소 보석이 공명하여 정화됨 | 매칭 시 빛의 파동 + 하프 소리 |
| **연쇄 매칭** | 공명이 연쇄적으로 확산되는 현상 | 콤보 숫자 + "Resonance Chain!" |
| **블록 소멸** | 정화된 보석이 순수한 에너지로 승화 | 반짝이며 사라지는 파티클 |

세계관: "관통의 의식" - 루나가 습득한 1단계 상급 기술

2. 정화된 공간에서 순수한 보석이 다시 자라남
3. 격자가 원래의 빛을 되찾음
4. 정화 점수(에너지 축적) 획득
```

### 6.5 스테이지 클리어의 의미

```
게임: 미션 목표 달성 → 스테이지 클리어
세계관: 한 지역의 헥사코어 그리드를 완전히 정화하여 빛을 되찾음

연출:
1. 마지막 오염체 제거 시 화면 전체가 빛남
2. "Region Purified!" 메시지
3. 지역 배경이 어둡고 오염된 모습에서 밝고 화려한 모습으로 변화
4. 루나가 기뻐하며 승리 포즈
5. 프리즘이 "우리가 해냈어!" 외침
```

### 6.6 턴 시스템의 의미

```
게임: 제한된 턴 수
세계관: 루나의 마법 에너지는 한정적이며, 효율적으로 사용해야 함

UI 연출:
- 턴 수 = "남은 에너지"
- 턴 소진 = "마법 에너지 고갈"
- 3턴 이하 = 루나가 지친 표정, 프리즘이 "조심해!" 경고

대사:
- 턴 5 이하: "에너지가 부족해지고 있어..."
- 턴 0: "마법 에너지가... 다했어..." (게임오버)
```

### 6.7 블록 티어 시스템의 의미

```
Normal → Tier1 → Tier2 → Tier3 → ProcessedGem

세계관 의미:
- Normal: 일반 보석 (기본 에너지)
- Tier1: 1회 정화된 보석 (에너지가 응축되기 시작)
- Tier2: 2회 정화된 보석 (에너지가 더욱 강해짐)
- Tier3: 3회 정화된 보석 (거의 순수한 상태)
- ProcessedGem: 완전히 정화된 보석 (빛나는 완벽한 크리스탈)

비주얼:
- 티어가 올라갈수록 보석이 더 크고 밝게 빛남
- ProcessedGem은 반짝이는 별 이펙트
```

## 7. 모바일 퍼즐 게임에 적합한 톤

### 7.1 전체 톤 설정

**밝고 희망찬 판타지 모험**
- 어둠과 위기가 있지만, 항상 희망과 해결의 빛이 있음
- 캐릭터들은 긍정적이고 응원하는 태도
- "위기 극복"보다는 "성장과 발견"에 초점

### 7.2 감정 곡선

```
챕터 1-3: 밝고 경쾌한 모험 (신나는 여정 시작)
챕터 4-6: 도전과 성장 (조금씩 어려워지지만 극복 가능)
챕터 7-8: 긴장과 시련 (진지해지지만 절망적이지 않음)
챕터 9-10: 긴박한 최종 결전 → 희망찬 대단원
```

### 7.3 짧은 세션 친화적 스토리텔링

- **1스테이지 = 1에피소드**: 각 스테이지마다 작은 이야기 완결
- **스킵 가능**: 반복 플레이 시 컷씬/대화 스킵 가능
- **요약 제공**: 오랜만에 접속한 플레이어를 위한 "지난 이야기" 요약
- **캐주얼 톤**: 대화는 가볍고 읽기 쉬운 문체

### 7.4 전연령 대상 배려

- **폭력성 제거**: "파괴"가 아닌 "정화"
- **무섭지 않은 적**: 오염체들도 귀엽거나 신비로운 디자인 가능
- **긍정적 메시지**: 협력, 용기, 희망, 성장 등 교육적 가치
- **친근한 캐릭터**: 루나와 프리즘은 어린이부터 성인까지 좋아할 수 있는 디자인

### 7.5 감정이입 요소

#### 루나와 플레이어의 동일시
- 루나는 "선택받은 영웅"이 아닌 "노력하는 평범한 소녀"
- 플레이어가 퍼즐을 풀수록 루나도 성장하는 구조
- 실패해도 "다시 도전하자!"는 긍정적 메시지

#### 조력자들의 응원
- 오라클리온: 따뜻한 격려
- 프리즘: 함께 기뻐하고 슬퍼하는 친구
- 실비아: 멋진 선배의 인정과 칭찬

### 7.6 스토리가 게임플레이를 방해하지 않도록

#### 스토리 분량 조절
- 컷씬: 최대 20초
- 대화: 최대 3-4줄
- 스킵 버튼 항상 제공
- 스토리 없이도 게임 진행 가능

#### 선택적 스토리 소비
- **Light Mode**: 스토리 최소화, 퍼즐만 즐김
- **Story Mode**: 모든 컷씬과 대화 감상
- 설정에서 변경 가능

#### 시즌 2: 새로운 위협
- 엔트로피는 사라졌지만, 우주 저편에서 새로운 적이 등장
- 새로운 보석 원소 추가 (예: 다이아몬드, 오팔 등)
- 루나가 이제는 선배 수호자로서 새로운 견습생을 가르침

#### 사이드 스토리
- 실비아의 과거 이야기 (외전 챕터)
- 오라클리온의 젊은 시절 (프리퀄)
- 각 오염체들의 탄생 비화

#### 이벤트 스테이지
- 계절 이벤트: "봄꽃 축제", "크리스탈 할로윈" 등
- 특별 챌린지: "엔트로피의 복수" 타임어택 모드

## 10. 결론

HexaPuzzle의 세계관은 **밝고 희망찬 판타지**로, 보석의 공명을 통해 세계를 정화한다는 직관적이고 아름다운 컨셉을 가지고 있습니다.

### 핵심 강점
✅ **게임플레이와 자연스럽게 연결**: 모든 메카닉이 세계관적 의미를 가짐
✅ **전연령 친화적**: 폭력적이지 않고 긍정적인 메시지
✅ **확장 가능성**: 시즌 업데이트, 외전, 이벤트 등 다양한 컨텐츠 가능
✅ **모바일 친화적**: 짧은 세션에도 스토리를 즐길 수 있음
✅ **감정이입 가능**: 루나라는 매력적인 주인공과 함께 성장

이 세계관은 단순한 퍼즐 게임을 **의미 있는 모험**으로 변화시키며, 플레이어에게 "나는 세계를 구하고 있다"는 만족감을 제공합니다.
