## 📖 개요

HexaPuzzle의 스토리 진행에 맞춘 **10개 챕터 기본 미션 설계**입니다.
각 미션은 루나의 성장 여정과 연계되며, 새로운 적군을 소개하고 게임 메카닉을 단계적으로 확장합니다.

---

## 🎮 미션 체계 구조

```
Chapter 1-2 (기초)          → 1~3번 미션 (적군 소개)
    ↓
Chapter 3-4 (성장)          → 4~6번 미션 (전술 확장)
    ↓
Chapter 5-8 (심화)          → 7~9번 미션 (고급 전략)
    ↓
Chapter 9-10 (최종)         → 10번 미션 (보스 격파)
```

## 🎯 10개 핵심 미션 상세 설계

### **1️⃣ 미션: 루나의 첫 발걸음 - 색상 도둑 소탕**

**스토리 배경**
- 루나가 크리스탈 숲에서 처음 적군을 마주침
- 오라클리온의 지도 아래 기본 전략을 배움

**미션 타입**: Enemy Removal (적군 제거 전문)
**대상 적군**: 색상도둑(Chromophage) 3마리 제거
**적용 스테이지**: 1~10 (Chapter 1: 크리스탈 숲)

**난이도**: ⭐☆☆☆☆

**핵심 학습 목표**
- 적군의 존재와 "색상 제거" 메카닉 이해
- 드릴 특수 블록으로 무페널티 제거 (매칭은 매칭 불가)
- 특수 블록의 활용 가치 체감

**게임 메카닉**
- 색상도둑이 붙은 블록은 매칭에서 제외됨 (무색 처리)
- 드릴/폭탄/레이저/도넛으로 제거 가능
- 제거 시 "+파괴 점수 획득"

**보상**
- 경험치 100 (기본값)
- "드릴 마스터" 배지 (특수 블록 인식 향상)
- 대사: 오라클리온 - "보네, 색상을 되찾았구나. 참 잘했어."

**UI 연출**
- 색상도둑 감지: 블록 상단에 회색 슬라임 아이콘
- 제거 이펙트: 슬라임이 흰색 에너지로 정화되며 사라짐
- 미션 진행도: "색상도둑 제거 3/3" 진행 바

---

### **2️⃣ 미션: 구속을 벗다 - 속박의 사슬 해방**

**스토리 배경**
- 루나의 회전 마법이 봉쇄되는 경험
- 주변 블록을 활용한 간접 전략의 중요성을 깨달음

**미션 타입**: Enemy Removal (회전 제약 상황)
**대상 적군**: 속박의 사슬(Chain Anchor) 5개 제거
**적용 스테이지**: 5~15 (Chapter 1-2: 크리스탈 숲 ~ 사파이어 호수)

**난이도**: ⭐⭐☆☆☆

**핵심 학습 목표**
- "회전 불가" 상태에서의 전술 수립
- 해당 블록이 포함되지 않은 다른 영역의 회전으로 우회 전략
- 제한된 조건에서의 창의적 문제 해결

**게임 메카닉**
- 사슬이 감긴 블록의 삼각형 클러스터는 회전 불가
- 인접한 다른 블록 클러스터의 회전은 정상 작동
- 일반 매칭 또는 특수 블록으로 제거

**보상**
- 경험치 150
- "회전 기술 습득" 배지 (전술 이해도)
- 대사: 프리즘 - "루나! 주변을 보는 법을 배웠어!"

**UI 연출**
- 사슬 감지: 블록 둘레에 검은 쇠사슬 애니메이션
- 터치 시도: "회전 불가" 경고음 + 붉은 X 플래시
- 제거 이펙트: 사슬이 산산조각 나며 자유 상태로 복원

### **3️⃣ 미션: 위험한 선택 - 가시 기생충의 함정**

**스토리 배경**
- 루나가 첫 번째 "위험한 딜레마"를 경험
- 일반 매칭의 비효율성 대비 특수 블록의 절대 가치 증명

**미션 타입**: Strategic Special Block Usage (특수 블록 우선)
**대상 적군**: 가시 기생충(Thorn Parasite) 제거 (드릴 금지, 폭탄/레이저만 사용)
**적용 스테이지**: 10~25 (Chapter 2-3)

**난이도**: ⭐⭐☆☆☆

**핵심 학습 목표**
- 특수 블록의 우선 사용 전략 수립
- "일반 매칭 제거 = 턴 -1 + 0점" vs "특수 블록 제거 = 무페널티 + 점수"
- 턴 관리의 극한 효율화

**게임 메카닉**
- 가시 기생충이 붙은 블록을 일반 매칭으로 제거하면:
  - **턴 -1 페널티** (추가 1턴 소모)
  - **파괴 점수 획득 안 함** (무점수)
- 폭탄/레이저로 제거하면 무페널티 + 점수 획득
- 드릴은 예외: 사용 금지 (미션 조건)

**미션 진행도 추적**
- 폭탄으로 제거: +1
- 레이저로 제거: +1
- 도넛으로 제거: +1
- 목표: 3개 제거

**보상**
- 경험치 200
- "특수 블록 전문가" 배지
- 폭탄 생성 확률 +10% (이후 스테이지)
- 대사: 실비아 - "좋은 선택이군. 진정한 수호자는 효율을 본다네."

**UI 연출**
- 가시 감지: 블록 상단에 붉은 가시 뭉치 (펄스 애니메이션)
- 일반 매칭 시도: "⚠️ 턴 -1, 점수 0점!" 경고 표시
- 특수 블록 제거: 가시가 정화 에너지에 녹아 사라짐

### **4️⃣ 미션: 확산의 위협 - 분열체 차단**

**스토리 배경**
- 루나가 "제거가 오히려 상황을 악화시킨다"는 역발상을 체험
- 전략적 우선순위 판단의 중요성 깨달음

**미션 타입**: Smart Enemy Removal (광역 특수 블록 강조)
**대상 적군**: 분열체(Divider) 3개 제거 (폭탄/레이저만, 드릴 금지)
**적용 스테이지**: 15~35 (Chapter 3-4)

**난이도**: ⭐⭐⭐☆☆

**핵심 학습 목표**
- "드릴 사용 = 분열 발생" → 폭탄/레이저 선택의 강제
- 광역 특수 블록(폭탄, 레이저)과 선택형 특수 블록(드릴) 구분
- 도넛의 절대적 우월성 체감 (분열 없이 완전 제거)

**게임 메카닉**
- 분열체가 붙은 블록 제거 시:
  - **드릴 사용**: 라인 제거 + 분열 발생 (인접 블록에 새 분열체)
  - **폭탄 사용**: 범위 제거 + 분열 없음
  - **레이저 사용**: 십자 제거 + 분열 없음
  - **도넛 사용**: 색상 전체 제거 + 분열 없음

**미션 완료 조건**
- 폭탄 또는 레이저로 분열체 3개 완전 제거 (드릴 사용 시 미션 실패)

**보상**
- 경험치 250
- "광역 제거 기술" 배지
- 보석 가공(Tier 상승) 보너스 +20% (이후 스테이지)
- 대사: 오라클리온 - "제거 방식도 실력의 일부란 것을 배웠군."

**UI 연출**
- 분열체 감지: 블록 위에 세포 모양 (청록색, 떨림)
- 드릴 사용 시도: "⚠️ 분열이 발생합니다!" 경고 + 빨간색 UI
- 폭탄/레이저 사용: 범위 내 모든 분열체가 동시에 정화
- 분열 이펙트: 분열체가 2개로 쪼개져 인접 블록으로 이동

### **5️⃣ 미션: 중력의 덫 - 왜곡자 격파 & 연쇄 달성**

**스토리 배경**
- 루나가 낙하 메카닉의 진정한 의미를 깨달음
- 연쇄(캐스케이드)와 낙하가 어떻게 상호작용하는지 학습

**미션 타입**: Dual Objective (적군 제거 + 연쇄 달성)
**목표 1**: 중력왜곡자(Gravity Warper) 2마리 제거
**목표 2**: 그 과정에서 5연쇄 달성 (최소 1회)
**적용 스테이지**: 20~40 (Chapter 3-4)

**난이도**: ⭐⭐⭐⭐☆

**핵심 학습 목표**
- 낙하 메카닉에 대한 깊이 있는 이해
- "중력 왜곡 = 낙하 차단" → 우선 제거 전략
- 연쇄 연쇄의 기쁨 (특수 블록 없이 자연 캐스케이드 경험)

**게임 메카닉**
- 중력왜곡자 제거 시:
  - 해당 블록의 낙하 차단 해제
  - 인접 블록의 느린 낙하도 정상화 (2배 속도 → 정상)
- 중력왜곡자 제거 후 자연스러운 낙하 → 연쇄 유도

**미션 추적**
- 목표 1 진행도: "중력왜곡자 제거 2/2"
- 목표 2 진행도: "최대 연쇄 깊이 5/5"
  - 한 턴 내 최대 연쇄 깊이 추적
  - 목표 달성 시 자동 완료

**보상**
- 경험치 300
- "연쇄 마스터" 배지
- 콤보 보너스 +15% (이후 스테이지)
- 대사: 프리즘 - "와! 보석들이 춤을 추고 있어! 이게 루나의 진정한 힘이야!"

**UI 연출**
- 중력왜곡자 감지: 블록 주변 보라색 소용돌이 (떠있는 애니메이션)
- 낙하 차단 시각화: 위 블록이 떠있는 상태로 애니메이션
- 제거 이펙트: 중력장이 사라지며 반짝이는 에너지 확산
- 연쇄 시작 시: "Cascade x5!" 텍스트 + 화려한 콤보 이펙트

### **6️⃣ 미션: 마법 방어막 강습 - 반사 장막 2단계 격파**

**스토리 배경**
- 루나가 엔트로피의 방어 기술에 대항할 방법을 찾아야 함
- "도넛 = 모든 것을 파괴하는 궁극의 기술"이라는 깨달음

**미션 타입**: High-tier Special Block Requirement (도넛 강제)
**대상 적군**: 반사 장막(Reflection Shield) 3개 제거 (도넛만 사용)
**적용 스테이지**: 25~50 (Chapter 4-5)

**난이도**: ⭐⭐⭐⭐☆

**핵심 학습 목표**
- 도넛 블록(7매칭+)의 생성 조건과 절대 가치 이해
- "반사 장막 = 일반 특수 블록 2회 필요" vs "도넛 = 1회 제거"
- 고난이도 특수 블록 조성의 보상 시스템

**게임 메카닉**
- 반사 장막의 기본 능력:
  - 드릴/폭탄/레이저 범위에 포함 → 장막만 파괴, 적군은 남음
  - 2번째 공격으로 적군 제거
  - 일반 매칭으로는 1회에 제거 가능 (이 미션에선 사용 금지)
- 도넛 사용 시: 장막 무시 → 1회 완전 제거 (색상 전체 정화)

**미션 완료 조건**
- 도넛 생성 후 반사 장막 3개를 도넛으로 제거
- 일반 매칭이나 다른 특수 블록 사용 시 진행도 리셋

**보상**
- 경험치 350
- "도넛 사냥꾼" 배지 (7매칭 이해도)
- 도넛 블록 생성 조건 5% 완화 (6.8매칭 이상 생성 가능)
- 대사: 실비아 - "마지막이 정답이었군. 진정한 수호자는 최강의 기술을 아낌없이 쓸 줄 안다."

**UI 연출**
- 반사 장막 감지: 블록 위 은색 육각형 방패 아이콘
- 일반 특수 블록 시도: "🛡️ 장막이 공격을 반사했다!" 텍스트 + 탄퀁 이펙트
- 도넛 사용: 무지개 에너지가 방패를 뚫고 장막 + 적군 동시 정화
- 제거 이펙트: 은빛 방패가 산산조각 나며 무지개 파동으로 흩어짐

### **7️⃣ 미션: 시간의 저주 극복 - 회전 제약 밀리초 게임**

**스토리 배경**
- 루나가 "시간 정지자"의 저주 속에서 극한의 효율을 보여야 함
- 턴 관리의 극한을 경험하는 터닝 포인트

**미션 타입**: Turn Management Extreme (회전 횟수 제한)
**대상 적군**: 시간 정지자(Time Freezer) 2마리 제거
**조건**: 총 회전 횟수 10회 이내로 달성
**적용 스테이지**: 30~60 (Chapter 5-6)

**난이도**: ⭐⭐⭐⭐☆

**핵심 학습 목표**
- "회전 = 턴 소모" 의 명확한 인식
- 시간 정지자가 있으면 **회전당 2턴** 소모 (1턴 추가 페널티)
- 각 회전을 3배 신중하게 생각하는 습관 형성

**게임 메카닉**
- 시간 정지자 제거 전: 해당 블록 포함 삼각형 회전 시 2턴 소모
- 터치 시점: "⚠️ 2턴 소모됩니다" UI 경고
- 시간 정지자 제거 후: 정상 1턴 복귀

**미션 추적**
- 우측 상단에 "회전 횟수: 5/10" 표시
- 회전 시 횟수 감소 (터치 피드백)
- 10회 초과 시: "⚠️ 회전 한도 초과" 경고 + 추가 회전 불가
- 적군 2개 제거 후 자동 완료

**보상**
- 경험치 400
- "턴 관리 대사" 배지
- 이동 아이템(턴 +1) 게임 내 구매 가능 해제
- 대사: 오라클리온 - "참. 나이가 들면서 배워야 할 첫 번째 미덕이 효율이지. 넌 배웠구나."

**UI 연출**
- 시간 정지자 감지: 블록 위 파란 시계 (초침 멈춘 모양, 떨림)
- 회전 불가 경고: 터치 시 "2턴!" 텍스트 + 빨간 플래시
- 회전 횟수 카운트: 우상단 "회전: 5/10" 숫자 애니메이션
- 제거 이펙트: 시계 바늘이 다시 움직이기 시작하며 정화

### **8️⃣ 미션: 쌍둥이 공명 동시 격파 - 전략적 최고점**

**스토리 배경**
- 루나가 "동시성"이라는 고급 전술을 배움
- 보드 전체를 읽고 계획하는 진정한 수호자로 성장

**미션 타입**: Dual Target Synchronized Removal (동시 제거)
**대상 적군**: 공명 트윈(Resonance Twin) 2쌍 (4개 블록) 정확히 3턴 내 제거
**조건**: 같은 턴 내에 각 쌍 모두 제거 (한 쌍만 제거하면 3초 후 부활)
**적용 스테이지**: 35~70 (Chapter 6-7)

**난이도**: ⭐⭐⭐⭐⭐

**핵심 학습 목표**
- 보드 전체 시야로 4개 목표 블록 추적
- 특수 블록 조합 활용 (폭탄 + 드릴, 레이저 + 폭탄 등)
- "동시성 = 전략의 절정"이라는 철학적 이해

**게임 메카닉**
- 공명 트윈은 항상 2개 쌍으로 등장 (인접 선 연결)
- 한쪽 제거 → 3초 후 반대쪽 부활 → 무한 루프
- **동시 제거 필수**: 같은 턴 내 제거하면 완전 소멸
- 도넛 또는 범위가 양쪽 포함 특수 블록(폭탄/레이저 + 폭탄) 조합 필요

**미션 추적**
- 좌측에 공명 트윈 쌍 시각화 (에너지선 표시)
- 턴 카운트: "남은 턴: 3/3" (매 턴마다 감소)
- 쌍 제거 진행도: "쌍 1/2 제거" → "완료!"

**보상**
- 경험치 500 (최고 보상)
- "공명 마스터" 배지 (보드 인식도 최고)
- 특수 블록 동시 생성 가능 이벤트 활성화 (같은 턴에 2개 특수 블록 생성 가능)
- 대사: 실비아 - "저것이 진정한 크리스탈 수호자의 경지야, 루나. 넌 이미 충분히 훌륭하다."

**UI 연출**
- 공명 트윈 감지: 같은 색의 2개 크리스탈 + 에너지선 (반짝임)
- 한쪽 제거: "⚠️ 다른 하나도 제거하세요!" 경고 + 타이머 표시 (3초)
- 동시 제거: 두 크리스탈이 동시에 무지개 빛으로 정화
- 완료 이펙트: 화면 전체가 밝게 빛나며 "Perfect Synchronization!" 메시지

### **9️⃣ 미션: 어둠의 감염 봉쇄 - 시간 압박 생존**

**스토리 배경**
- 루나가 "방치 불가능한 위협"을 처음 경험
- 긴급 상황에서의 판단과 신속한 행동의 중요성

**미션 타입**: Time-Pressure Containment (확산 차단)
**대상 적군**: 어둠의 포자(Shadow Spore) 초기 1개 제거
**조건**: 포자의 확산을 최대 3개까지만 허용 (4개 이상 확산 시 미션 실패)
**적용 스테이지**: 40~80 (Chapter 7-8)

**난이도**: ⭐⭐⭐⭐⭐

**핵심 학습 목표**
- "방치 = 자동 실패"라는 긴박감
- 우선순위 판단 (포자 제거 vs 다른 목표)
- 광역 특수 블록(도넛, 레이저)의 시간 압박에서의 활용

**게임 메카닉**
- 어둠의 포자:
  - 3턴마다 인접 블록에 새 포자 생성 (자동)
  - 포자가 2개 이상 있으면 확산 주기 단축 (2턴, 1턴)
  - 중첩 시 보드 전체 오염 가능성
- 제거: 해당 블록 매칭 또는 특수 블록 (도넛 1회에 전체 제거)

**미션 추적**
- 상단에 "포자 개수: 1/3" 카운터
- 포자 확산 타이머: 다음 확산까지 "3턴 남음" 표시
- 4개 생성 시: "⚠️ 감염 봉쇄 실패!" → 미션 자동 실패

**미션 완료 조건**
- 초기 포자 1개 제거 (그 과정에 확산된 포자도 함께 제거 가능)
- 포자 4개 이상 확산 전 완료

**보상**
- 경험치 450
- "감염 차단자" 배지
- 자동 포자 정화 아이템 1회 제공 (즉시 모든 포자 제거)
- 대사: 오라클리온 - "시간과의 싸움이야. 빠른 판단이 생명이지."

**UI 연출**
- 어둠의 포자 감지: 블록을 어둡게 처리 + 검은 연기 이펙트
- 확산 발생: "확산!" 텍스트 + 어둠이 인접 블록으로 번지는 애니메이션
- 타이머 경고: 확산까지 1턴 남았을 때 붉은 경고음
- 제거 이펙트: 포자가 밝은 빛으로 정화되며 보드가 원래 색상 복귀

### **🔟 최종 미션: 카오스 군주 격파 - 모든 기술의 종합**

**스토리 배경**
- 루나의 최종 시험: 엔트로피의 화신을 정면 대결
- 모든 배운 메카닉을 한 턴에 녹여내는 클라이막스

**미션 타입**: Boss Battle (종합 미션)
**대상 적군**: 카오스 군주(Chaos Overlord) 1마리 격파
**특수 조건**: 동시에 3가지 세부 목표 달성 필수
**적용 스테이지**: 60~100 (Chapter 8-10, 최종 보스 스테이지)

**난이도**: ⭐⭐⭐⭐⭐⭐ (특별 난이도)

**카오스 군주의 기본 메카닉**
- 매 턴 **랜덤 2~3가지 효과** 적용:
  - 색상 제거 (Chromophage)
  - 회전 불가 (Chain Anchor)
  - 낙하 차단 (Gravity Warper)
  - 특수 블록 무효화 1회 (Reflection Shield)
- 예측 불가능한 변수 → 적응적 전략 필수

**세부 목표 (3가지 동시 달성)**

**목표 A**: 카오스 군주를 도넛으로 격파
- 도넛 생성 필수 (7매칭 이상)
- 도넛 사용하면 군주의 모든 효과 무시하고 1회 제거

**목표 B**: 격파 과정에서 10 콤보 달성
- 자연 연쇄 또는 특수 블록 조합으로 10연쇄 이상 한 번 달성
- 보드의 모든 메카닉을 조화시키는 경험

**목표 C**: 3가지 세부 목표 모두 달성
- 목표 A + 목표 B + **턴 제한 내 격파** (스테이지별 턴 제한 준수)
- 시간 압박 속 완벽한 플레이의 증명

**미션 진행도**
```
[카오스 군주 격파]
├─ 목표 A: 도넛 생성 및 사용
├─ 목표 B: 최대 10 콤보 달성 (진행도 바)
└─ 목표 C: 턴 제한 내 완료
```

**게임 내 표시**
- 상단: "카오스 군주 HP: ███████░░░" (3단계 약화 시스템)
- 좌측: "현재 효과: 색상제거, 낙하차단" (매 턴 변경)
- 우측: "도넛 생성 진행도" (매칭 카운트)

**보상 (엄청남)**
- 경험치 1000 (매우 높음)
- "궁극의 수호자" 배지 (최고 명예)
- **무한 미션 모드 '생존 도전' 활성화** (새로운 게임 모드 언락)
- 특수 블록 생성 확률 영구 +25%
- 대사: 오라클리온 - "이제 넌 진정한 크리스탈 수호자다, 루나. 세계의 운명이 바뀌었느니라."
- 대사: 프리즘 - "루나! 우리가 해냈어! 이제 누군가 우릴 도와줄 필요가 없어!"
- 대사: 실비아 - "축하한다, 루나. 이제 넌 내 동료다."

**UI 연출**
- 카오스 군주 감지: 거대한 무지개색 혼돈의 오라 (왕관 형상)
- 턴 시작: "다음 효과: ___" 팝업 + 붉은 경고음
- 도넛 생성: 7매칭 달성 시 "도넛 준비 완료!" 금색 플래시
- 도넛 사용: 무지개 에너지가 군주를 감싸며 정화 (화려한 이펙트)
- 격파 완료: 화면 전체 밝음 + "CHAOS VANQUISHED!" 대문자 텍스트 + 승리 음악 (30초)

**엔딩 시네마틱 (스테이지 100 클리어 직후)**
```
1. 카오스 군주가 정화되며 사라짐
2. 크리스탈리아 전체가 밝은 빛으로 복구
3. 루나가 고향 마을로 돌아옴
4. 마을 사람들이 축제로 환영
5. 오라클리온이 "진정한 크리스탈 수호자" 칭호 수여
6. 프리즘과 루나가 웃으며 다음 모험 암시
7. "The End... or a New Beginning?" (후속 시즌 암시)
```
