# HexaPuzzle 적군(Enemy) 시스템 기획서

**작성일:** 2026-02-15
**작성자:** 기획 디렉터
**문서 버전:** v1.0

---

## 1. 개요

HexaPuzzle에 내러티브와 전략적 깊이를 부여하기 위한 **적군(Enemy) 시스템**입니다. 매 턴마다 또는 특정 조건에서 블록 위에 적군이 등장하여 플레이어의 퍼즐 플레이를 방해하며, 다양한 적군 조합을 통해 난이도와 전략성을 단계적으로 확장합니다.

---

## 2. 핵심 메카닉

### 2.1 적군 생성 규칙
- **생성 타이밍**: 매 N턴마다 또는 특정 이벤트 트리거 시
- **생성 위치**: 랜덤 블록 또는 특정 패턴 (스테이지별 설정)
- **생성 개수**: 스테이지 진행도에 따라 1~5개
- **제한 사항**: 이미 적군이 붙은 블록, 특수 블록(드릴/폭탄 등)에는 중복 생성 불가

### 2.2 적군 제거 조건
- **매칭**: 적군이 붙은 블록을 매칭에 포함시켜 제거
- **특수 블록**: 드릴/폭탄/레이저/도넛/X블록의 범위에 포함
- **연쇄**: 매칭 연쇄(Cascade)에서도 제거 가능
- **제거 실패 페널티**: 일부 적군은 제거 실패 시 진화/확산

### 2.3 UI 표시
- **블록 위 오버레이**: 작은 적군 아이콘 (육각형 블록 상단 20% 영역)
- **테두리 강조**: 적군이 있는 블록은 붉은색/회색 테두리
- **애니메이션**: 점멸, 흔들림, 펄스 효과
- **제거 이펙트**: 적군 파편, 스크린 플래시, 스코어 팝업

## 3. 적군 종류 (10가지)

### 3.1 기본 적군 (1~3번)

---

#### 1. 색상 도둑 (Chromophage)

**시각적 특징**
- 블록 위에 작은 회색 슬라임 형태
- 블록 색상을 흡수하여 회색/무채색으로 변경
- 슬라임이 블록을 뒤덮는 듯한 반투명 오버레이
- 미세한 펄스 애니메이션 (색상 빨아들이는 느낌)

**방해 효과**
- **색상 제거**: 블록의 GemType은 유지하지만 시각적으로 회색 처리
- **매칭 불가**: 해당 블록은 다른 같은 색 블록과 매칭되지 않음 (MatchingSystem에서 제외)
- **회전 가능**: 삼각형 회전은 정상 작동 (회전 중 다른 블록과 위치 교환 가능)
- **티어 유지**: 블록의 티어는 그대로 유지

**제거 조건**
- 특수 블록(드릴/폭탄/레이저/도넛/X블록) 범위에 포함
- 아이템(색상 복원 아이템) 사용 시 제거

**등장 시점**: Stage 1~10 (튜토리얼 단계)

**난이도 등급**: ★☆☆☆☆

**전략적 의미**
- 초보자에게 "적군 = 방해물"이라는 개념 도입
- 특수 블록의 중요성 강조
- 회전은 가능하므로 완전히 막히진 않음

---

#### 2. 속박의 사슬 (Chain Anchor)

**시각적 특징**
- 블록을 감싸는 검은 쇠사슬 이미지
- 블록 중앙에 작은 자물쇠 아이콘
- 회전 시도 시 사슬이 팽팽해지는 애니메이션
- 금속성 질감, 어두운 회색~검은색

**방해 효과**
- **회전 불가**: 해당 블록이 포함된 삼각형 클러스터는 회전 불가 (InputSystem에서 터치 무시)
- **색상 유지**: 블록의 색상과 매칭 기능은 정상
- **고립 위험**: 주변 블록만 회전 가능하므로 매칭 기회 감소

**제거 조건**
- 해당 블록을 매칭에 포함시켜 제거
- 특수 블록(드릴/폭탄/레이저/도넛/X블록) 범위에 포함
- 아이템(사슬 해제 아이템) 사용

**등장 시점**: Stage 5~15

**난이도 등급**: ★★☆☆☆

**전략적 의미**
- 회전 전략의 중요성 강조
- 주변 블록을 활용한 간접 매칭 유도

---

#### 3. 가시 기생충 (Thorn Parasite)

**시각적 특징**
- 블록 위에 작은 붉은 가시 뭉치
- 블록 표면에서 가시가 자라나는 듯한 모션
- 주기적으로 가시가 펄스 (위협적인 느낌)
- 진한 빨강~보라색 그라데이션

**방해 효과**
- **제거 시 데미지**: 해당 블록을 매칭으로 제거하면 **턴 -1 페널티**
- **색상/회전 정상**: 모든 기본 기능은 정상 작동
- **리스크-리턴**: 제거하면 페널티, 방치하면 보드 공간 점유

**제거 조건**
- 특수 블록(드릴/폭탄/레이저/도넛/X블록)으로 제거 시 페널티 없음
- 아이템(가시 제거제) 사용 시 페널티 없이 제거

**등장 시점**: Stage 10~25

**난이도 등급**: ★★☆☆☆

**전략적 의미**
- "제거하면 손해"라는 역발상 적군
- 특수 블록 우선 사용 유도
- 턴 관리의 중요성 강조

---

### 3.2 중급 적군 (4~6번)

---

#### 4. 분열체 (Divider)

**시각적 특징**
- 블록 위에 작은 세포 형태 (분열 중인 모습)
- 반투명 청록색/녹색 액체
- 미세하게 떨리는 애니메이션
- 제거 시 2개로 쪼개지는 이펙트

**방해 효과**
- **분열 복제**: 이 적군이 붙은 블록을 매칭으로 제거하면, **인접한 빈 블록 중 1개에 새로운 분열체 생성**
- **회전/색상 정상**: 기본 기능 정상 작동
- **확산 위협**: 방치하면 보드 전체에 확산 가능성

**제거 조건**
- 특수 블록(폭탄/레이저/도넛) 범위에 포함 시 분열 없이 완전 제거
- 드릴: 분열체가 라인에 여러 개 있어도 모두 제거하지만 분열 발생
- 아이템(바이러스 백신) 사용 시 분열 없이 제거

**등장 시점**: Stage 15~35

**난이도 등급**: ★★★☆☆

**전략적 의미**
- "제거가 오히려 확산을 부름" 긴장감
- 특수 블록의 전략적 선택 유도 (폭탄/레이저 우선)
- 빠른 처리 vs 신중한 계획 딜레마

---

#### 5. 중력 왜곡자 (Gravity Warper)

**시각적 특징**
- 블록 주변에 보라색 소용돌이 이펙트
- 블록이 약간 위아래로 떠다니는 듯한 애니메이션
- 주변 공간이 일그러지는 왜곡 효과
- 어두운 보라~검은색 그라데이션

**방해 효과**
- **낙하 차단**: 이 적군이 있는 블록은 **낙하 처리(ProcessFalling)에서 제외**
- 위에 빈 공간이 있어도 블록이 떠 있음
- **주변 영향**: 직접 인접한 블록도 낙하가 느려짐 (낙하 애니메이션 2배 느림)
- **색상/회전 정상**

**제거 조건**
- 해당 블록 매칭 또는 특수 블록으로 제거
- 아이템(중력 정상화) 사용

**등장 시점**: Stage 20~40

**난이도 등급**: ★★★★☆

**전략적 의미**
- 낙하 메카닉을 방해하여 연쇄 차단
- 보드 정리 타이밍 지연
- 우선 제거 대상 판단력 요구

---

#### 6. 반사 장막 (Reflection Shield)

**시각적 특징**
- 블록 위에 육각형 방패 아이콘
- 은빛 금속성 광택
- 특수 블록 공격 시 튕겨내는 플래시 이펙트
- 얇은 에너지 장벽 같은 느낌

**방해 효과**
- **특수 블록 무효화**: 드릴/폭탄/레이저/X블록의 범위에 포함되어도 제거되지 않음 (장막만 깨짐)
- **1회 보호**: 특수 블록 1회 공격 시 장막만 파괴, 2번째 공격에서 적군 제거
- **도넛 예외**: 도넛 블록(전체 색상 제거)은 1회에 제거 가능
- **색상/회전 정상**

**제거 조건**
- 특수 블록 2회 공격 또는 도넛 1회
- 일반 매칭으로는 1회에 제거 가능
- 아이템(장막 파쇄기) 사용 시 즉시 제거

**등장 시점**: Stage 25~50

**난이도 등급**: ★★★★☆

**전략적 의미**
- 특수 블록의 "만능성"을 제한
- 공격 순서와 타이밍 전략 필요
- 일반 매칭의 가치 재발견

---

### 3.3 고급 적군 (7~10번)

---

#### 7. 시간 정지자 (Time Freezer)

**시각적 특징**
- 블록 위에 파란 시계 아이콘 (초침이 멈춘 모습)
- 블록 주변에 얼음 결정 파티클
- 차가운 청백색 광채
- 주기적으로 시계 초침이 떨리는 애니메이션

**방해 효과**
- **회전 카운트 증가**: 이 적군이 있는 블록이 포함된 삼각형을 회전하면 **2턴 소모** (1턴 추가 페널티)
- **색상/매칭 정상**
- **경고 표시**: 터치 시 "2턴 소모 경고" UI 표시

**제거 조건**
- 해당 블록 매칭 또는 특수 블록으로 제거
- 아이템(시간 가속기) 사용

**등장 시점**: Stage 30~60

**난이도 등급**: ★★★★☆

**전략적 의미**
- 턴 관리의 극한 도전
- 회전 전략 재고 (정말 이 회전이 2턴 가치인가?)
- 특수 블록 우선 제거 유도

---

#### 8. 공명 트윈 (Resonance Twin)

**시각적 특징**
- 항상 **2개 한 쌍**으로 생성
- 블록 위에 같은 색상의 작은 쌍둥이 크리스탈
- 두 적군 사이에 얇은 에너지선 연결
- 주기적으로 빛이 왕복하는 애니메이션
- 하나를 선택하면 다른 하나도 강조 표시

**방해 효과**
- **연동 제거**: 한 쪽만 제거하면 **즉시 다른 쪽이 부활** (3초 후 재생성)
- **동시 제거 필수**: 같은 턴 내에 두 블록을 모두 제거해야 완전 소멸
- **색상/회전 정상**

**제거 조건**
- 한 턴 안에 두 블록을 모두 매칭/특수 블록으로 제거
- 도넛 블록 또는 폭탄(범위가 양쪽 모두 포함)이면 한 번에 제거 가능
- 아이템(공명 차단기) 사용 시 연동 해제 후 개별 제거 가능

**등장 시점**: Stage 35~70

**난이도 등급**: ★★★★★

**전략적 의미**
- "동시성" 전략 요구 (한 턴에 복수 목표 달성)
- 보드 전체를 보는 시야 훈련
- 특수 블록 조합 활용 (폭탄+드릴 등)

---

#### 9. 어둠의 포자 (Shadow Spore)

**시각적 특징**
- 블록 위에 검은 포자 구름
- 블록이 어둡게 보이며 주변까지 어두워짐
- 제거되지 않으면 매 3턴마다 검은 연기가 피어오름
- 불길한 보라~검정 그라데이션

**방해 효과**
- **확산 타이머**: 3턴마다 **인접한 빈 블록 또는 일반 블록 중 1개에 새로운 포자 생성**
- **중첩 가능**: 여러 포자가 있으면 확산 속도 가속 (2턴마다, 1턴마다...)
- **색상/회전 정상**
- **보드 점령**: 방치하면 보드 전체가 포자로 뒤덮임

**제거 조건**
- 해당 블록 매칭 또는 특수 블록으로 제거
- 레이저/도넛처럼 광범위 공격이 효과적
- 아이템(정화의 빛) 사용 시 모든 포자 동시 제거

**등장 시점**: Stage 40~80

**난이도 등급**: ★★★★★

**전략적 의미**
- 시간 압박 (방치 불가)
- 우선순위 판단 (포자 먼저 vs 미션 먼저)
- 광역 특수 블록의 중요성

---

#### 10. 카오스 군주 (Chaos Overlord)

**시각적 특징**
- 블록 위에 왕관 쓴 악마 형상
- 무지개색 혼돈의 오라
- 블록 전체가 격렬하게 떨림
- 제거 시 폭발적인 화면 이펙트
- 보스급 위압감

**방해 효과**
- **복합 저주**: 다음 효과 중 **랜덤 2~3개를 동시 적용**
  - 색상 제거 (Chromophage)
  - 회전 불가 (Chain Anchor)
  - 낙하 차단 (Gravity Warper)
  - 특수 블록 무효화 1회 (Reflection Shield)
- **변화**: 매 턴마다 적용 효과가 바뀜 (예측 불가)
- **보드당 1개 제한**: 동시에 2개 이상 생성 안 됨

**제거 조건**
- 도넛 블록 또는 X블록으로만 제거 가능 (고급 특수 블록 필수)
- 일반 매칭/드릴/폭탄/레이저는 1회 공격 시 효과 1개만 제거 (3회 공격 필요)
- 아이템(신성한 빛) 사용 시 즉시 제거

**등장 시점**: Stage 60~100 (보스 스테이지)

**난이도 등급**: ★★★★★★ (특별 난이도)

**전략적 의미**
- 모든 적군 메카닉의 종합판
- 최상위 특수 블록 생성 유도 (도넛/X블록)
- 극한의 전략과 순발력 요구
- "보스전" 느낌의 클라이막스

## 4. 시스템 구현 가이드

### 4.1 데이터 구조 확장

```csharp
// BlockData.cs에 추가
public enum EnemyType
{
    None = 0,
    Chromophage = 1,      // 색상 도둑
    ChainAnchor = 2,      // 속박의 사슬
    ThornParasite = 3,    // 가시 기생충
    Divider = 4,          // 분열체
    GravityWarper = 5,    // 중력 왜곡자
    ReflectionShield = 6, // 반사 장막
    TimeFreezer = 7,      // 시간 정지자
    ResonanceTwin = 8,    // 공명 트윈
    ShadowSpore = 9,      // 어둠의 포자
```

### 4.2 적군 생성 시스템

```csharp
// EnemySpawnSystem.cs (신규)
public class EnemySpawnSystem : MonoBehaviour
{
    [SerializeField] private HexGrid hexGrid;
    [SerializeField] private int spawnInterval = 3; // N턴마다 생성

    private int turnCounter = 0;

    public void OnTurnEnd()
    {
        turnCounter++;
        if (turnCounter >= spawnInterval)
        {
            SpawnEnemy();
            turnCounter = 0;
        }
    }

    private void SpawnEnemy()
    {
        // 현재 스테이지에 맞는 적군 타입 선택
        EnemyType type = SelectEnemyType(currentStage);

        // 생성 가능한 블록 찾기
        List<HexBlock> candidates = FindSpawnCandidates();

        if (candidates.Count > 0)
        {
            HexBlock target = candidates[Random.Range(0, candidates.Count)];
            AttachEnemy(target, type);
        }
    }
}
```

### 5.2 밸런스 조정 포인트

- **생성 주기**: 너무 자주 생성되면 압박 과다, 너무 느리면 긴장감 부족
- **제거 난이도**: 특수 블록 생성 난이도와 적군 방어력 균형
- **페널티 강도**: ThornParasite 턴 감소, TimeFreezer 턴 배수 조정
- **확산 속도**: Divider/ShadowSpore 타이머 조정

## 9. 성공 지표

- **플레이어 참여도**: 적군 등장 스테이지의 평균 플레이 시간 20% 증가
- **특수 블록 사용률**: 특수 블록 생성 및 사용 빈도 30% 증가
- **난이도 만족도**: 스테이지 클리어율 60~70% 유지 (너무 쉽지도 어렵지도 않게)
- **재도전률**: 적군 등장 스테이지 재도전 횟수 증가 (도전 의욕)
