## 고블린 타입별 스탯

### 기본 고블린 (몽둥이)
| 항목 | 값 | 코드 위치 |
|------|-----|----------|
| HP | **6** | GoblinSystem.cs `SpawnGoblins()` 기본값 |
| 색상 | 녹색 (0.3, 0.65, 0.2) | `CreateGoblinSprite()` |
| 무기 | 몽둥이 | 근접 공격 |
| 등장 스테이지 | 1~15 전 스테이지 | |

### 궁수 고블린 (활) — 원거리 고정형
| 항목 | 값 | 코드 위치 |
|------|-----|----------|
| HP | **1** | GoblinStageConfig `archerHp = 1` |
| 색상 | 보라색 (0.45, 0.25, 0.65) | `CreateArcherGoblinSprite()` |
| 눈동자 | 노란색 (0.9, 0.7, 0.1) | 날카로운 인상 |
| 무기 | 활 + 화살 | 왼쪽에 활, 오른쪽에 화살 |
| 등장 스테이지 | 12~15 | archerRatio로 확률 소환 |
| **소환 위치** | **가장 윗줄만** (rMin - 3) | `SpawnGoblins()` topRowCoords |
| **이동** | **불가** (MoveAllGoblins에서 제외) | `!g.isArcher` 필터 |
| **공격 패턴** | **2턴마다** 안 깨진 블록에 화살 발사 | `ArcherAttackPhase()` |
| **화살 효과** | 블록 1회 크랙 (근접 고블린과 동일) | `ApplyArcherDamageToBlock()` |
| **화살 비주얼** | 포물선 궤적 + 0.3초 페이드아웃 | `FadeArrowTrajectory()` |
| **대기 모션** | 좌우 ±3도 흔들림 (0.3초) | `ArcherIdleAnimation()` |
| **낙하 면역** | 낙하 블록에 데미지 안 받음 | `ApplyIndividualFallDamage` 궁수 체크 |
| **취약점** | 드릴/폭탄/드론 등 특수 블록만 데미지 가능 | |
| **드론 우선** | 드론(+모든 콤보)이 궁수를 1순위 타겟 | `FindBestArcherTarget()` |

### 갑옷 고블린 (방패) — 근접 탱커형
| 항목 | 값 | 코드 위치 |
|------|-----|----------|
| HP | **15** (기본보다 높음) | GoblinStageConfig `armoredHp = 15` |
| 색상 | 진한 녹색 (0.25, 0.5, 0.2) + 은색 갑옷 | `CreateArmoredGoblinSprite()` |
| 장비 | 은색 투구 + 흉갑 + 방패 + 몽둥이 | |
| 투구 장식 | 금색 세로줄 | |
| 등장 스테이지 | 12~15 | armoredRatio로 확률 소환 |
| 이동 | 기본 고블린과 동일 (블록 방향 우선) | |
| 공격 | 근접 공격 (기본과 동일) | |

### 공통 속성 (기본 + 갑옷)
| 항목 | 값 |
|------|-----|
| 이동 방향 | 블록 있는 방향 우선, 없으면 아래(0,1) |
| 공격 패턴 | 이동 전 블록 공격 → 파괴 → 이동 |
| 블록 공격 단계 | 정상→금간(isCracked)→껍데기(isShell)→파괴 |
| 사망 조건 | HP ≤ 0 (낙하 충돌 데미지) |

## 고블린 소환 조건

- **활성화 범위**: `GameManager.cs` → `selectedStage >= 1 && selectedStage <= 15`
- **고블린 시스템 비활성 시**: `goblinSystem.CleanupAll()` 호출
- **첫 턴**: 게임 시작 시 즉시 `ProcessTurn()` → 고블린 소환
- **이후 턴**: 캐스케이드 완료 → `SpawnEnemiesViaSystemAndPlay()` → `goblinSystem.ProcessTurn()`
